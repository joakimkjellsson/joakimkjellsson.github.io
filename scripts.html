Stream functions in various coordinates

This is the code that was used for the Kjellsson et al. (2014) and Kjellsson (2015) papers. It is written in Fortran 90 with OpenMP shared-memory parallelisation and netCDF4 input/outputs. It also uses CDO to convert or interpolate data where necessary. It reads in ERA-Interim and CMIP5 atmosphere data to calculate stream functions in a wide range of different coordinate systems.

Barotropic, linear shallow water model with MPI

This code was written by me and Abubakr Salih for a project on high-performance computing. We used it to test scalability and performance on various supercomputers and compilers. netCDF module was written by Laurent Brodeau.
